# Music Master

Music Master is still in production.

## Requirements
- [Visual Studio 2022](https://visualstudio.com)
- [Vulkan SDK](https://vulkan.lunarg.com/sdk/home#windows)

- [GLM](https://github.com/g-truc/glm)
- [pytube](https://github.com/pytube/pytube)
